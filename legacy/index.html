Why This Matters

Some stories disappear before anyone knows to ask.

A parent dies before a child is old enough to remember their voice. A grandparent’s recipe never gets written down. A family story changes every time it gets retold. LYLO exists to help preserve the words, voice, and memories directly from the person who lived them.

What Happens Later

The memory stays protected when life changes.

LYLO is built for the moments families do not always know how to prepare for. Memories can be saved, permissioned, protected, and later shared only according to the person’s choices.

Approved memories can be shared

Locked memories stay locked

Legacy Vault

Every memory needs permission.

LYLO is designed around memory boundaries. Some stories are meant for family. Some are private. Some are meant to be released later. The person recording stays in control.

Governed By Truth

Do not invent someone’s life.

What LYLO must do

  • Use stored memories as evidence
  • Mark uncertainty clearly
  • Respect private vault boundaries
  • Ask instead of guessing

What LYLO must never do

  • Make up memories
  • Pretend uncertainty is fact
  • Share locked memories without permission
  • Replace the real person with a fake version
